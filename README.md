# Expense Tracker (Streamlit App)

A modern and interactive **Expense Tracker Web App** built using **Streamlit**, **Pandas**, and **Matplotlib**.  
This app helps users manage their income, track expenses, analyze spending patterns, and download reports easily.

---

## Features

- **Dashboard**
  - View total income, expenses, and savings
  - Monthly filtering
  - Visual insights (Pie chart & Trend graph)

- **Add Expense**
  - Add daily expenses with category, amount, and description

- **View Expenses**
  - Display all expenses in a table
  - Download data as CSV file

- **Analysis**
  - Category-wise expense analysis using bar chart

- **Category Management**
  - Add custom categories dynamically

- **Modern UI**
  - Gradient background
  - Glassmorphism card design
  - Clean and responsive layout

---

## Tech Stack

- **Frontend & Backend**: Streamlit  
- **Data Handling**: Pandas  
- **Visualization**: Matplotlib  
- **UI Components**: streamlit-option-menu
